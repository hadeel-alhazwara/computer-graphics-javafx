# JavaFX Animation & Computer Graphics Projects 🎨

## 📌 Description

This repository contains a collection of interactive computer graphics and animation projects developed using JavaFX. The projects demonstrate animation techniques, drawing using canvas, and building interactive educational applications.

## 🎬 Projects Included

### 🌊 Sea Animation

* Animated sea waves using mathematical functions (sin waves)
* Moving ship with dynamic motion
* Birds animation in the sky
* Gradient sky and water effects
* Sun reflection on water

### 🌴 Island Scene

* Sunset sky with gradient colors
* Animated water waves
* Palm trees and island drawing
* Moving birds
* Ship animation

### ⚡ Pikachu Animation

* Fully drawn character using shapes (Path, Ellipse, Polygon)
* Smooth animations (jumping, mouth movement)
* Sound effects using MediaPlayer
* Advanced effects (DropShadow, GaussianBlur)

### 🧒 Kids Learning App (Arabic Letters)

* Educational application for children
* Helps kids learn Arabic letters interactively
* Simple UI and engaging visuals

## 🚀 Features

* Real-time animation using Timeline and AnimationTimer
* Canvas-based drawing
* Mathematical wave simulation
* Interactive graphics
* Audio integration
* Clean and structured code

## 🛠️ Technologies Used

* Java
* JavaFX
* Animation (Timeline, AnimationTimer)
* Graphics (Canvas, Shapes)
* MediaPlayer (for sound)

## ▶️ How to Run

1. Open the project in your IDE (IntelliJ / Eclipse)
2. Make sure JavaFX is configured
3. Run any main class:

   * Art.java
   * Sea_art1.java
   * Pikachu.java

## 🎯 Purpose

This project was created as part of a university coursework to explore computer graphics, animation, and interactive application development.
